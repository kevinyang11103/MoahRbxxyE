# 抑郁症患者看护系统简介 python610

## 项目概述

本项目是一款基于Python的抑郁症患者看护系统，采用Django框架进行后端开发，前端使用Vue.js进行构建，数据库选用MySQL 5.7。

## 技术栈

- **后端：**
  - Python 3.8
  - Django
- **前端：**
  - Vue.js
- **数据库：**
  - MySQL 5.7

## 功能模块

- 患者信息管理
- 健康数据监测
- 治疗进度跟踪
- 报告生成与导出

## 系统角色

- 系统管理员
- 医护人员
- 患者

## 运行环境

- Python 3.8及以上版本
- Django框架
- MySQL 5.7数据库
- Vue.js前端框架

## 部署步骤

1. 环境准备：确保Python 3.8、MySQL 5.7已正确安装。
2. 数据库配置：根据提供的SQL脚本初始化数据库。
3. 代码部署：将后端Django代码部署到服务器。
4. 前端部署：将Vue.js构建的前端代码部署到Web服务器。

## 目录结构

```
project/

├── frontend/             # 前端Vue.js项目目录

```
